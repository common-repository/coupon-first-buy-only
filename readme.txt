Adds an option to apply coupon only on the first client order.

== Description ==

Adds a new checkbox in new and edit coupon in WooCommerce, which when checked will only allow use the coupon on the first order.

If you have any questions or difficulties in using the plugin, please visit the Support section in [this link](https://wordpress.org/support/plugin/coupon-first-buy-only).

**Alert:** _you have to disable guest checkout in WooCommerce, otherwise plugin will not know if is the first user order_!

== Installation ==

= Plugin installation: =

* Send the unziped files to the folder "wp-content/plugins", or install using the WordPress plugins manager.
* Activate the plugin.
